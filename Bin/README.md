Gnuwin32 tools to compile all the dependencies and the IDE.

Make for Windows: http://gnuwin32.sourceforge.net/packages/make.htm<br>
Binaries: http://gnuwin32.sourceforge.net/downlinks/make-bin-zip.php (make.exe)<br>
Dependencies: http://gnuwin32.sourceforge.net/downlinks/make-dep-zip.php (libiconv2.dll, libintl3.dll)

CoreUtils for Windows: http://gnuwin32.sourceforge.net/packages/coreutils.htm<br>
Binaries: http://gnuwin32.sourceforge.net/downlinks/coreutils-bin-zip.php (cp.exe, pwd.exe, rm.exe, touch.exe)<br>
Dependencies: coreutils-dep-zip not required, already exist in make-dep-zip and more up to date

Zip for Windows: http://gnuwin32.sourceforge.net/packages/zip.htm<br>
Binaries: http://downloads.sourceforge.net/gnuwin32/zip-3.0-bin.zip (zip.exe)<br>
Dependencies: http://downloads.sourceforge.net/gnuwin32/zip-3.0-dep.zip (bzip2.dll)
