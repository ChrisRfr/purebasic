# GNU Dependencies for Windows

This folder contains the Windows version of some GNU tools required to build the PureBasic IDE, taken from the [GnuWin] project on SourceForge.

# Make for Windows

The following files were taken from the [Make for Windows] v3.81 package ([`make-3.81-bin.zip`][make-3.81-bin.zip] and [`make-3.81-dep.zip`][make-3.81-dep.zip]), released under the terms of the [GPLv2 license][Make_COPYING]:

- [`Make_COPYING`][Make_COPYING] — GPLv2 license text.
- `make.exe`
- `libiconv2.dll`
- `libintl3.dll`

# CoreUtils for Windows

The following files were taken from the [CoreUtils for Windows] v5.3.0 package ([`coreutils-5.3.0-bin.zip`][coreutils-5.3.0-bin.zip]), released under the terms of the [GPLv2 license][coreutils_COPYING]:

- [`coreutils_COPYING`][coreutils_COPYING] — GPLv2 license text.
- `cp.exe`
- `pwd.exe`
- `rm.exe`
- `touch.exe`

# Zip for Windows

The following files were taken from the [Zip for Windows] v3.0 package ([`zip-3.0-bin.zip`][zip-3.0-bin.zip] and [`zip-3.0-dep.zip`][zip-3.0-dep.zip]), released under the terms of the [Info-ZIP license][zip_LICENSE]:

- [`zip_LICENSE`][zip_LICENSE] — Info-ZIP license text.
- `zip.exe`
- `bzip2.dll`
